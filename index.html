4th Year Project

Cloud-based Secure File Sharing System: The aim of my project is to address the growing need for secure and efficient remote collaboration. By integrating TrueNAS, Nextcloud, and Seafile, the platform provides a comprehensive solution that prioritizes the core principles of information security: confidentiality, integrity, access control, and availability. TrueNAS ensures robust and scalable backend storage, Nextcloud enables real-time collaboration and synchronization, and Seafile delivers high-performance file synchronization. Together, these technologies create a secure, flexible, and user-friendly system for remote access, advanced encryption, and multi-layered user authentication, empowering organizations to protect sensitive data while enhancing productivity in a remote work environment.
