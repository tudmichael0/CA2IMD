Bootstrap Features Overview

Here are some of the key Bootstrap classes and components used in my portfolio website:

  • Navbar:
    • navbar, navbar-expand-lg, navbar-light, bg-light: Creates a responsive navigation bar with a light background.
    • navbar-toggler, navbar-toggler-icon: Button to toggle the navbar on smaller screens.
    • navbar-collapse, navbar-nav, nav-item, nav-link, ms-auto: Controls the collapsing behavior and alignment of the navbar items.
  • Hero Section:
    • bg-primary, text-white, text-center, py-5: Applies background color, text color, text alignment, and padding.
    • container: Centers the content and provides padding.
  • Button (Modal Trigger):
    • btn, btn-primary: Applies Bootstrap's button styles.
    • d-flex, justify-content-center, mt-5, mb-4: Aligns the button centrally with margin spacing.
  • Modal:
    • modal, fade, modal-dialog, modal-content: Basic modal structure and animation.
    • modal-header, modal-body, modal-footer: Sections of the modal for the header, body, and footer.
    • btn-close: Close button style for the modal.
  • Carousel:
    • carousel, carousel-inner, carousel-item: Creates a carousel component with slides.
    • carousel-control-prev, carousel-control-next: Navigation controls for the carousel.
    • d-block, w-100, rounded: Ensures images are displayed properly with full width and rounded corners.
    • carousel-control-prev-icon, carousel-control-next-icon: Icons for the carousel controls.
  • Accordion:
    • accordion, accordion-item, accordion-header, accordion-button, accordion-collapse: Used to create an accordion that expands and collapses.
    • accordion-body, d-flex, justify-content-center, align-items-center: Flexbox layout within the accordion body for content alignment.
  • Grid System:
    • container, row, col-md-8, col-md-12: Bootstrap grid system to structure content in rows and columns, providing responsive layout.
  • Footer:
    • bg-dark, text-white, text-center, py-3: Footer with a dark background, white text, centered content, and padding.
  • Responsive and Utility Classes:
    • ms-auto: Aligns the navbar items to the right.
    • text-center: Centers text.
    • py-5, mt-4, mb-4, mt-5: Adds padding and margin to various sections.
  • Image Classes:
    • rounded-circle, img-fluid: Applies a circular shape to the profile image and ensures responsive scaling of images.
